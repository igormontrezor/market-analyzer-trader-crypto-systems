Sistema de trading Igor Montrezor — método interativo passo a passo

O método opera no alinhamento de timeframes maiores com execução refinada nos menores. Clique em cada passo para explorar.

Passo 1
Identificar tendência
Supertrend + direção no timeframe maior. Nunca operar contra a tendência.
Passo 2
Canal RSI no TF maior
RSI no suporte/resistência do canal. Peso maior para semanal e mensal.
Passo 3
Stoch RSI no TF menor
Stochastic RSI no contra-fundo/topo — inverso da tendência nos TFs menores.
Passo 4
Fibonacci + R:R mín 1:5
0.5–0.618 tocado. Take profit na extensão 0.618 (A+) ou 0.328 (A/B).
Passo 5
Executar entrada
Inversão do Supertrend. Stop no extremo do canal RSI. Máx 3% risco.
Passo 6
Gerenciar saída
Take profits via Fibonacci/Neuro Athena. Alerta RSI + Stoch em direções opostas.
Identifique a tendência principal usando o Supertrend no timeframe maior (semanal/mensal). A tendência é o maior peso do sistema — nunca opere contra ela.

O canal do RSI é a base do sistema. O preço precisa estar no suporte ou resistência do canal para acionar uma entrada válida.

Lógica de confirmação
Compra: Tendência semanal de alta → esperar Stoch RSI no 4H/D1 no fundo (oversold) → entrada quando virar para cima.

Venda: Tendência semanal de baixa → esperar Stoch RSI no 4H/D1 no topo (overbought) → entrada quando virar para baixo.

O Stoch RSI está sempre na direção contrária à tendência no TF menor — isso é o pullback confirmado.
