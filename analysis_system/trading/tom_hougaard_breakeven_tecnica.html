Técnica de Breakeven de Tom Hougaard — Passo a Passo

Método Hougaard
Breakeven + Pirâmide · Checklist operacional
Os três pilares
Proteção
Mover para BE elimina risco financeiro da unidade inicial.
Pirâmide
Com a base protegida, adiciona contratos ao que já funciona.
Corte rápido
Se o mercado volta ao BE, sai sem hesitar. Tese falhou no timing.
Passo a passo
01
Entrada inicial
"Estou testando uma ideia."
Tamanho de posição controlado. Risco definido antes de entrar. Esta é apenas a sonda — não o bet principal.
risco fixo
02
Mercado anda a favor
"Talvez eu esteja certo."
Observar confirmação de momentum. Aguardar distância suficiente baseada em volatilidade ou nível técnico próximo. Não agir ainda.
paciência
03
Mover stop para breakeven
"Agora posso pressionar."
Stop da posição 1 vai para o preço de entrada. A operação agora é de risco zero. Este é o gatilho que libera o próximo passo — não apenas proteção de ego.
stop → entradarisco liberado
04
Adicionar posição (piramidagem)
"Vou aumentar no que funciona."
Abre posição 2. O stop combinado é gerenciado de forma que, se o mercado reverter, o lucro da P1 no BE compensa a perda da P2. Nunca adicionar em operação perdendo.
posição 2 abertastop combinado
05
Mercado continua — maximizar assimetria
"Maximizar assimetria."
Deixar os vencedores correr. Subir stops progressivamente conforme o mercado avança. Repetir o ciclo para posição 3 se as condições permitirem.
trailing stopdesconforto produtivo
06
Reversão ao breakeven — sair
"A tese falhou no timing."
Sem hesitação, sem esperança. O retorno ao BE é sinal de que o setup não se desenvolveu como esperado. Fechar tudo. Preservar capital mental para a próxima.
saída imediatazero ego
A frase
"Pessoas normais buscam conforto. Traders de sucesso buscam desconforto. Mover para o breakeven é o primeiro passo para poder dobrar a aposta."
— Tom Hougaard
Armadilhas documentadas
Stop cedo demais
Mover para BE antes da distância necessária expõe a operação ao ruído normal do mercado. Você sai antes do movimento real acontecer.
BE como muleta emocional
Usar o BE apenas para "não perder o que ganhou" é viés de aversão à perda disfarçado. Para Hougaard, o BE é ferramenta de expansão — não de conforto.
